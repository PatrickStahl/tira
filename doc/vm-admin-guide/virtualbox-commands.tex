List vms on host pc:\\
  \cmd{VBoxManage list vms}\\
\medskip\noindent  
List running vms on host pc:\\
  \cmd{VBoxManage list runningvms}\\
\medskip\noindent  
Start vm:\\
  \cmd{VBoxManage startvm <uuid>|<name> --type headless}\\
\medskip\noindent  
Stop vm :\\
  \cmd{VBoxManage controlvm <uuid>|<vm-name> savestate}\\
\medskip\noindent  
Delete vm:\\
  \cmd{VBoxManage unregistervm <uuid>|<vm-name> --delete}\\
\medskip\noindent  
Import ova-file:\\
  \cmd{VBoxManage import <ova>}\\
\medskip\noindent
